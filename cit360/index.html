Learning and Teaching:

As I mentioned above, the discussion boards were a great place for resources and to work with the class. Another thing about this class that we did was work with a group. Sometimes this can be hard to do remotely, but at my work we have meetings everyday where someone is remote, including myself. I think that this is becoming more and more common in the workplace. This class can prepare you for remote working. When you work remote, which for me is working from home, you have to be self-motivated and focused on your work. It could be easy to get distracted at home with kids or things that you want to get done around the house, but you must be disciplined. Just like in this class. You must be responsible for your own learning and getting your work done.

In this class, and in the workplace, your team can make or break you. I learned again that having a good team is vital to succeeding. The opposite is true as well, if you don't have a good team you are more prone to fail. Being part of a team means that you are there to help one another and, as done in this class, you teach one another. Java Topics:

I have taken a few Java classes now, and this one had been the most interesting in a lot of ways. Some classes have been more in depth on different topics but this one went over fourteen topics that if you had the time, you could spend multiple semesters to really cover them all. I was a bit overwhelmed with all that was required for this class and even though I didn't do a final project, I can see the value is making this class so intense.

I had to humble myself a little bit and get past the frustration. It wasn't until I changed my attitude and sat down and did the work that I was actually able to think through it and get it done. In the end, I think that this class in its own way was good for me to take. It helped me to see a new way to think when it comes to a college course and with how I manage my time.
