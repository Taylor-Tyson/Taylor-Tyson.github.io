CSS Class Properties Using Javascript

With JavaScript there is the ability to modify classes and attributes for HTML elements. I am going to show a couple examples on how this works. We will start with modifying an attribute for am image tag.

Modifying Attributes

When working with attributes, they are usually a name value pair. Most common attributes are the src and img tags, but there are many more. There are four methods that can be used to modify the attributs.

  • hasAttribute()
  • getAttribute()
  • setAttribute()
  • removeAttribute()
