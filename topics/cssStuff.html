CSS Transitions, Animations, and Transformations

On this page we are going to talk about and show examples of using CSS to transition, animate, and transform property values of an HTML element.

CSS Transitions

CSS transitions allow you to change the property values of an element smoothly and in a given duration of time. In the example below you can hover over the image and see the transition to a new imgage. The tranistion time is set to four seconds which is kind of long but I wanted to show the transition longer.

Animations

In short, CSS animations allow you to animate elements in HTML without using Flash or JavaScript. That is good because some browsers are not going to support Flash much longer. The thing to make note of when using animations is the keyframes rule. You have to bind an animation to a keyframe which will run the animation gradually from the current style to what you want it to be.
