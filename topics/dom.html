Document Object Model (DOM)

The document object model (DOM) is techincally an API. Using the DOM allows you to be able to get, change, add, or delete HTML elements on an HTML page. DOM is not a programming language but allows JavaScript, or other languages, to in essence call the DOM and make changes.

In this example we are going to use JavaScript to append li objects to a list. You will also have the option to remove the last object added in case you make a mistake. Enter a name or text and click on the "Add" button to add it to the list. If you need to remove your entry click on "Undo" button.
