JavaScript Events

Simply put as stated on w3schools.com, events are just things that happen to HTML elements. You can use JavaScript to create and define what happens when events happen. There are a lot of HTML events that you can work with. Some of the more common ones are:

  • Onchange
  • Onclick
  • Onmouseover
  • TouchEvent

For a more defined list, visit: here. I will be doing examples of just a few of the ones listed above.

Onchange

You can use the onchange event to call a function when a value of an element changes. For our example we are going to make a selection which will notify us what was selected.

Onclick

This one I have used a lot on these pages. I have used the onclick to many times to call a function in JavaScript when a button is pressed. In this simple example we are going to click the "Click Me" button which will call a function that will alert us that we clicked the button.
