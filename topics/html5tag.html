HTML5 Video Tag

The HTML5 video tag allows you to play a video on your web page without the need for a seperate plug-in. With the video tag you can embed a MP4, WebM, or Ogg formats. Below is a video that is embedded by using the video tag.

The HTML for the tag is as shown below. Notice the controls option which will give the user the controls to work with the video. Without this option then there would be no way to start the video, unless you did auto play or other ways to control the video. You can also create your own controls using javascript. 

    
    <video id="video1" width="475" controls>
        <source src="/media/big_buck_bunny.mp4" type="video/mp4">
        <source src="/media/big_buck_bunny.ogv" type="video/ogg">
        Your browser does not support HTML5 video.
    </video>

HTML5 Audio Tag

The audio tag is similar to the video tag in that it allows you to embed audio on your page without using any plug-ins or codecs. You can use MP3, WAV, or OGG forkate. As with the video tag there are some attributes that you can use. Such as:

  • Autoplay - Plays audio file after the page loads.
  • Controls - Displays the control panel.
  • Loop - Repeats the audio file.
  • Muted - Mutes the sound when the audio file is played.
  • SRC - Specifies the path to the audio file.

Below is an audio file that you can play using the audio as well as the HTML:

    
    <audio controls>
        <source src="/media/computer_trusting.wav"type="audio/wav">
        Your browser does not support the audio tag.
    </audio>

HTML5 Canvas Tag

The cancas tag allows you to add graphics and to your page that run on the fly. You can use the canvas tag to draw text, draw graphics, to create games, and this can all be animated or interactive.
