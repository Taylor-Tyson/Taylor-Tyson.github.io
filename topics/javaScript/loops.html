Loops

Thare are five different kinds of loops that you can use in JavaScript. Loops are used to execute the same block of code more than once. They are good for automoting repetitive tasks in your program. The loops that we are going to talk about are:

  • while loop
  • do-while loop
  • for loop
  • for-in loop
  • for-of loop

One importnat thing to mention with using loops is that you need to be carefule that you don't code them in a way that creates an infinite loop. This will cause your program and computer major issues!

While Loop

The while loop is a loop that will exeucte while a variable is set at a certain value. In our example we are going to click the count down button and count down from five using a while loop.

When you click the button it is calling a function that contains the following:

            let i = 5;

Do-while Loop

The do-while loop is similar to using the while loop. The main difference I could find between the two is that when using the while loop, the code in the loop will only be executed if the condition is in a true state. If it is not, then the code will not be executed. With a do-while loop, it will always run the code in the loop once and stop when the condition is in a false state.

In this example when you click the count up button it will count up from zero to five.

When you click the button it is calling a function that contains the following:

        
        function countUp() {
            let i = 0;
            let myText = "";
                
            do {
                myText += "< br>" + i;
                i++;
            }
            while (i <= 5);
            document.getElementById("cntUp").innerHTML
            = myText;
        }
