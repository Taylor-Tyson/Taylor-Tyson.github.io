JavaScript Objects

Objects in JavaScript are an important part of using JavaScript. To me it seems that objects in JavaScript are somewhat confusing at first, but when you start to understnad them they are easy to work with. In realiity, almost everything in JavaScript are objects. Here is a list of some common objects:

  • Boolens (If defined correctly)
  • Numbers (If defined correctly)
  • Strings (If defined correctly)
  • Dates
  • Arrays
  • Functions

One thing that you can also do is create your own object. In the next couple examples I am going to show how you can do that in different ways.

JavaScript Class

Classes in JavaScript are a type of function that are declared with the keyword class. The properties of the class are assigned inside of a constructor method. In this example we are going to be using a Truck class which was created as follows: 

        
        export class Truck {
            constructor(manufacturer, model, year){
                this.manufacturer = manufactuer;
                this.model = model;
                this.year = year;
            }
        }
