JSON

JSON stands for JavaScript Object Notation. It is a syntax for storing and exchanging data. When you exchange data from a browser and a server, the data can only be text. Here is an example of how JSON string should be formatted as string values:

{"make":"Ford","model":"Super Duty","year":2019}

For JavaScript the string values can be written with double or single quotes:

{make:'Ford', model:'F-350', year:2019}

Notice that numbers there are no quotes. Numbers must be an integer or floating point.

There are two methods for JSON. The parse and stringify methods.

JSON.parse()

The parse method will parse a string and return it as a Javascript object. This will only work if the string is written in the JSON format as shown above. We will use the JSON string above to show and example of how the parse method works.

When you push the parse button it calls a function in JavaScript that is written as: 

        
        function jsonParseDemo () {
            var obj = JSON.parse(document.getElementById
            ("string").innerHTML);

            document.getElementById("demo").innerHTML = 
            obj.make + ", " + obj.model + ", " + obj.year;
        }

JSON.stringify()

The JSON stringify method works opposite of the parse method, in that it will take a JavaScript object and turn it into a string. When this is used, the data sent to the web server needs to be a string. To show this, let's take the JavaScript object that was returned from the parse method and turn it back to a string.

When you push the stirngify button it calls a function in JavaScript that is written as: 

        
        var myObj = {"make":"Ford","model":"Super Duty"
        ,"year":2019};

        function jsonStringifyDemo () {
            var myText = JSON.stringify(myObj);
            document.getElementById("demo1").innerHTML
            = myText;
        }
